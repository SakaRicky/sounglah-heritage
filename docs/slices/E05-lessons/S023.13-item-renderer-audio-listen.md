# S023.13 - Lesson Item Renderer: AUDIO_LISTEN

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Planned

## Goal

Listening-first screen: large play control, optional hidden text until the learner chooses to show it.

## Dependencies

- S023.11 (audio player).
- S023.10 (player shell).

## UI Contract

- Heading from item title or default “Listen carefully” (i18n).
- `instructionText` if provided.
- **Large central play button** for Médumba audio.
- When `contentJson.hideTextUntilPlayed` is true (default):
  - Hide Médumba text and translations initially.
  - After first play (or always available — product choice): show **Show text** button to reveal phrase + EN/FR.
- When `hideTextUntilPlayed` is false: show text alongside player.

### Continue button

**Do not disable Continue** until audio plays (3-year-old UX). Show encouraging copy (“Listen with grandma!”); **Show text** appears after first play when `hideTextUntilPlayed` is true. Continue is always enabled.

## Out Of Scope

- Dictation exercise.
- Multiple choice after listen.

## Acceptance Criteria

- Greeting Grandma item 4 works with `hello_grandma` concept audio.
- Show text toggle works.
- Accessible play button (label, focus).
- Lint/build pass.

## Verification

Manual: listen step with grandma; reveal text.

```bash
cd frontend
npm run lint
npm run build
```
