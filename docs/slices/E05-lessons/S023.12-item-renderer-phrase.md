# S023.12 - Lesson Item Renderer: PHRASE

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Planned

## Goal

Render a full reusable phrase with Médumba text, audio, translations, and optional usage note.

## Dependencies

- S023.11 (shared concept payload pattern).
- S023.10 (player shell).

## UI Contract

- Phrase as hero text (Médumba).
- Play audio (Médumba).
- English and French translations.
- **Usage note** from `contentJson.usageNote` when present — smaller supportive text (e.g. when to say it with grandma).
- Optional `instructionText` from item.

Example for Greeting Grandma item 3: phrase concept `hello_grandma`.

## Design Notes

- Visually distinct from VOCABULARY: phrase may span two lines; slightly larger type.
- Keep layout calm — one primary action (listen + read).

## Out Of Scope

- Speak-back / speech recognition.
- Multiple audio variants picker.

## Acceptance Criteria

- PHRASE type replaces placeholder in player.
- Usage note displays when set.
- Shares `conceptPayload` resolution with VOCABULARY ([overview contract](./00-overview.md#public-play-payload-contract)).
- Lint/build pass.
