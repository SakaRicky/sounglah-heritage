# S023.9 - Public Lesson Intro Screen

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Done

## Goal

Prepare the learner (child + grandma) before the first item — set expectations, show time and step count, one clear **Start lesson** action.

## Dependencies

- S023.8 (listing links here).
- Public lesson detail API.

## Backend

```text
GET /api/lessons/:slug
```

Response shape: see **Public Play Payload Contract** in [00-overview.md](./00-overview.md#public-play-payload-contract).

Intro page calls `GET /api/lessons/:slug` and uses **metadata only** (title, description, cover, `estimatedMinutes`, `items.length`). It may ignore `conceptPayload` on each item — the player refetches the same endpoint on mount (S023.10).

404 for unknown slug or non-published lesson.

## Route

```text
/lessons/:slug
```

## UI Contract

- Large cover image.
- Lesson title (Playfair-friendly heading).
- Description paragraph.
- Chips or lines: estimated time, number of steps (`items.length`).
- Optional difficulty label for parents.
- **Start lesson** button → `/lessons/:slug/play`.
- Back link to `/lessons`.

Tone: welcoming, calm, child-safe — not gamified streak language.

## Out Of Scope

- Player navigation.
- Resolving concept text/audio on intro (player loads full payload).

## Acceptance Criteria

- Published Greeting Grandma intro renders correctly.
- Draft lesson URL returns 404 or friendly not-found.
- i18n EN/FR.
- Lint/build pass.

## Verification

Manual: open `/lessons/greeting-grandma`, confirm copy and Start button.

```bash
cd frontend
npm run lint
npm run build
```
