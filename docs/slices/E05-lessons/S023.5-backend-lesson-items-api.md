# S023.5 - Backend Lesson Items API

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Planned

## Goal

CRUD and reorder lesson items under a lesson, with type-specific validation for the four MVP item types.

## Dependencies

- S023.1 (models).
- S023.2 (lessons exist).

## API Contract

| Method | Path | Behavior |
| --- | --- | --- |
| GET | `/api/admin/lessons/:lessonId/items` | Ordered list of items for builder |
| POST | `/api/admin/lessons/:lessonId/items` | Create item; assign `order_index` if omitted (append) |
| PATCH | `/api/admin/lesson-items/:id` | Update fields |
| DELETE | `/api/admin/lesson-items/:id` | Hard delete; confirm in UI. Reorder remaining items or leave gaps per backend policy (recommend compacting `order_index` after delete). |
| PATCH | `/api/admin/lesson-items/:id/reorder` | Body: `{ "direction": "up" \| "down" }` — swap with adjacent item in same lesson |

### Item JSON shape

```json
{
  "id": "uuid",
  "lessonId": "uuid",
  "type": "VOCABULARY",
  "conceptId": "uuid",
  "title": "Grandma",
  "instructionText": null,
  "contentJson": {},
  "orderIndex": 1,
  "isActive": true,
  "createdAt": "...",
  "updatedAt": "..."
}
```

### Type-specific validation

| Type | `conceptId` | `contentJson` |
| --- | --- | --- |
| VOCABULARY | Required | Optional |
| PHRASE | Required | Optional `usageNote` string |
| AUDIO_LISTEN | Required | Optional `hideTextUntilPlayed` boolean (default true) |
| CULTURAL_NOTE | Optional | Required `noteTextEn` and `noteTextFr`; optional `imageUrl`, `imageAltText` |

- Reject link to disabled or missing concept for concept-backed types.
- `order_index` unique per lesson.

### Optional enrichment on GET (admin)

Include lightweight concept summary for picker display:

```json
"concept": {
  "id": "...",
  "key": "grandmother",
  "completionStatus": "published"
}
```

Reuse S021 completion service if practical; otherwise defer to S023.7.

## Out Of Scope

- Public lesson payload with resolved texts/audio (S023.8–S023.9).
- Bulk import from CSV.

## Acceptance Criteria

- All item endpoints authenticated.
- Cannot attach VOCABULARY without `conceptId`.
- CULTURAL_NOTE saves with `noteTextEn` and `noteTextFr`.
- Reorder returns updated list; after DELETE, compact `order_index` values (1…n).
- Pytest coverage for validation and reorder.
