# S023.4 - Admin Create/Edit Lesson Screen

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Planned

## Goal

Let admins create and edit lesson metadata, including the shell for **Greeting Grandma**, with save draft and publish actions.

## Dependencies

- S023.2 (lessons API).
- S023.3 (navigation from list).

## Routes

```text
/admin/content/lessons/new
/admin/content/lessons/:id/edit
```

## Form Fields

| Field | Required | Notes |
| --- | --- | --- |
| Title | Yes | |
| Slug | Yes | Auto-generate from title on create; editable |
| Description | No | Textarea |
| Difficulty | Yes | Select: beginner, intermediate, advanced (same as concepts) |
| Estimated minutes | No | Number input |
| Cover image | No | Upload via Cloudinary if pattern exists; else URL fields |
| Cover image alt text | No | Required when cover present (a11y) |
| Status | Yes | draft / published / archived |
| Order index | No | Integer for curriculum sort |

## Actions

- **Save Draft** — `status = draft` (or keep current status if editing published lesson cautiously).
- **Publish** — set `status = published`; disabled with inline reasons when:
  - zero active lesson items, or
  - any active concept-backed item links an unpublished concept.
  Mirror backend 400 messages from [publish guards](./00-overview.md#publish-guards-backend--admin-ui).
- **Cancel** — return to list.

## Validation (client + server)

- Title and slug required.
- Slug uniqueness surfaced from API 409.
- Warm error copy, not generic SaaS tone.

## Out Of Scope

- Lesson items editor (S023.6).
- Public preview player.

## Acceptance Criteria

- Publish button shows disabled state + reason list before submit when guards fail.
- Admin can create “Greeting Grandma” with slug `greeting-grandma`, beginner, 3–5 min estimate, draft status.
- Admin can edit and save existing lesson.
- Cover image upload works if Cloudinary wired for concepts (reuse component).
- i18n EN/FR for labels and buttons.
- Lint/build pass.

## Verification

Manual: create Greeting Grandma lesson; appears on list from S023.3.

```bash
cd frontend

npm run build
```
