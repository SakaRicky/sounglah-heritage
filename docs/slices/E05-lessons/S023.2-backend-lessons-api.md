# S023.2 - Backend Lessons API

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Done

## Goal

Expose authenticated admin CRUD for lessons so S023.3–S023.4 can list, create, and edit lessons without direct DB access.

## Dependencies

- S023.1 (models and migration).

## API Contract

Prefix: `/api/admin/lessons` — same auth middleware as other admin routes.

| Method | Path | Behavior |
| --- | --- | --- |
| GET | `/api/admin/lessons` | Paginated list; filters: `search`, `status`, `difficulty`; sort by `order_index`, `title`, `updated_at` |
| GET | `/api/admin/lessons/:id` | Single lesson; include `itemCount` (active items) if cheap to compute |
| POST | `/api/admin/lessons` | Create; require `title`, `slug`; default `status=draft`, `order_index=0` |
| PATCH | `/api/admin/lessons/:id` | Update editable fields; enforce publish guards when `status = published` |
| DELETE | `/api/admin/lessons/:id` | Hard delete; **CASCADE** deletes all lesson items. Return confirm-worthy 204. Prefer `status = archived` when hiding without data loss. |

### Request/response shape (camelCase JSON)

**Lesson**

```json
{
  "id": "uuid",
  "title": "Greeting Grandma",
  "slug": "greeting-grandma",
  "description": "Learn how to greet grandma with love and respect.",
  "difficulty": "beginner",
  "estimatedMinutes": 5,
  "coverImageUrl": null,
  "coverImagePublicId": null,
  "coverImageAltText": null,
  "status": "draft",
  "orderIndex": 0,
  "itemCount": 0,
  "createdAt": "...",
  "updatedAt": "..."
}
```

### Validation

- `title` required, trimmed.
- `slug` required, unique, URL-safe (reuse slug helper from concepts if present).
- `difficulty` ∈ `beginner`, `intermediate`, `advanced` (same as concepts).
- `estimatedMinutes` positive integer or null.
- `status` ∈ `draft`, `published`, `archived`.
- When setting `status = published`: require ≥1 active lesson item and every active concept-backed item links a published concept (see parent overview publish guards).

## Out Of Scope

- Lesson items endpoints (S023.5).
- Public `GET /api/lessons` (S023.8).
- Cover image upload (can stub URL fields; wire Cloudinary in S023.4 if concept upload pattern exists).
