# S023.3 - Admin Lessons List Screen

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Done

## Goal

Give admins a home base to see all lessons, filter by status/difficulty, and jump to create, edit, or manage items.

## Dependencies

- S023.2 (lessons API).

## Route

Prefer consistency with existing content admin:

```text
/admin/content/lessons
```

Replace S010 placeholder (`nextSlice="S014"`) with real list page.

## UI Contract

- **Page title:** Lessons (i18n EN/FR).
- **Subtitle:** Short family-centered line — e.g. “Build short lessons children can do with family.”
- **Primary action:** Create Lesson → `/admin/content/lessons/new` (or modal if that matches Concepts pattern).
- **Filters:** Status (`draft`, `published`, `archived`), difficulty, search by title/slug.
- **Layout:** Responsive cards or table (match Concepts/Languages admin style).

### Each lesson row/card shows

- Cover image thumbnail (placeholder if none).
- Title.
- Difficulty badge.
- Estimated minutes.
- Status badge.
- Active item count (from API `itemCount`).
- **Actions:** Edit, Manage Items, Preview (Preview can link to public intro in new tab when published; disabled or draft warning otherwise).

### States

- Loading skeleton.
- Empty state with CTA to create first lesson.
- Error state with retry.

## API Client

- Add `frontend/src/features/lessons/` (or align with existing feature folder layout).
- Types mirroring S023.2 JSON.
- `fetchLessons`, hooks for list page.

## Out Of Scope

- Create/edit form (S023.4).
- Items builder (S023.6).
- Cover upload UI.

## Acceptance Criteria

- Authenticated admin sees lessons from API.
- Filters and search work.
- Manage Items links to `/admin/content/lessons/:lessonId/items` (route may 404 until S023.6 — OK if linked early).
- EN/FR strings in i18n dictionaries.
- `npm run lint` and `npm run build` pass.

## Verification

```bash
cd frontend
npm run lint
npm run build
```

Manual: log in as admin, open lessons list, confirm empty and populated states.
