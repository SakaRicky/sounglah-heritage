# Sounglah Frontend

React + Vite + TypeScript + Tailwind frontend for Sounglah.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Structure

```text
public/
  images/        static assets (e.g. hero-family.png) served at /
eslint.config.js
vite.config.ts
tailwind.config.js
postcss.config.js
src/
  vite-env.d.ts
  app/
  pages/
  components/
  lib/
  styles/
```
