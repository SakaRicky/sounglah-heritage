# S023.10 - Lesson Player Shell

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Review

## Goal

Shared learner shell for stepping through lesson items: progress, back, previous/continue/finish — with **placeholders** per item type until S023.11–S023.14 ship.

## Dependencies

- S023.9 (entry from Start lesson).
- `GET /api/lessons/:slug` from S023.8 (full play payload with `conceptPayload`).

## Route

```text
/lessons/:slug/play
```

### Step navigation (locked)

- Current step index lives in **React state only** (`useState(0)`).
- **No `?step=` query param**, no `sessionStorage`, no backend progress.
- Browser refresh **restarts the lesson** at step 1 — acceptable for anonymous child+grandma MVP.

## UI Shell

- **Top:** Back to intro (`/lessons/:slug`).
- **Progress:** “1 / 5” text + thin progress bar.
- **Main area:** Renders current item via `LessonItemRenderer` switch on `type`.
- **Bottom nav:**
  - Previous (disabled on first step).
  - Continue (advances; on last step label **Finish** → warm completion message, then link to intro or `/lessons`).

### Placeholder renderers (this slice only)

Each type shows a simple card: type name + title + “Renderer coming in S023.1x” — enough to walk the full sequence.

## Data Loading

- Fetch `GET /api/lessons/:slug` once on mount (same payload as intro; intro page may use a lighter fetch later — not required for MVP).
- Items already filtered to active and sorted by backend.
- Handle empty items array gracefully.

## Out Of Scope

- Real VOCABULARY/PHRASE/AUDIO_LISTEN/CULTURAL_NOTE UI (S023.11–S023.14).
- Persisting completion across sessions.
- Hard-gating Continue on audio played (see S023.13).

## Acceptance Criteria

- Can step through all five Greeting Grandma placeholders.
- Progress indicator accurate.
- Refresh resets to step 1.
- Mobile layout usable with grandma holding the phone.
- i18n EN/FR for chrome strings (Continue, Finish, Back).
- Lint/build pass.
