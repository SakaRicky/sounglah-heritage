# S023.7 - Admin Create/Edit Lesson Item Screen

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Planned

## Goal

Dynamic admin form to create and edit each MVP lesson item type, with concept search and completion warnings.

## Dependencies

- S023.5 (items API).
- S023.6 (builder navigation).
- S021 completion API (for concept badges).

## Routes

```text
/admin/content/lessons/:lessonId/items/new
/admin/content/lessons/:lessonId/items/:itemId/edit
```

## Common Fields

| Field | Notes |
| --- | --- |
| Type | Select on create; read-only on edit |
| Title | Required |
| Instruction text | Optional learner prompt |
| Concept | Searchable picker — required except CULTURAL_NOTE |
| Order index | Optional; default append |
| Active | Toggle `isActive` |

## Type-Specific Fields

### VOCABULARY

- Concept required.
- No extra `content_json` fields for MVP.

### PHRASE

- Concept required (phrase-level concept such as `hello_grandma`).
- **Usage note** → `contentJson.usageNote`.

### AUDIO_LISTEN

- Concept required.
- **Hide text until played** checkbox → `contentJson.hideTextUntilPlayed` (default on).

### CULTURAL_NOTE

- Concept optional / hidden.
- **Note text (English)** → `contentJson.noteTextEn` (required).
- **Note text (French)** → `contentJson.noteTextFr` (required).
- Optional image URL + alt text in `contentJson`.

## Concept Picker

- Search by concept key/title.
- Show per-language badges: EN ✅/⚠️, FR ✅/⚠️, MED ✅/⚠️ (from completion workflow).
- Warn (non-blocking) if concept incomplete or unpublished; show completion badges EN/FR/MED.
- **Publish guard (read-only hint):** if lesson is published and item adds unpublished concept, show banner on lesson edit — backend blocks re-publish.

Link out to Concept Texts / Completion dashboard for fixes.

## Out Of Scope

- Live learner preview.
- Creating concepts inline (link to Concepts CRUD).

## Acceptance Criteria

- All four types can be created and edited.
- Validation messages match backend rules.
- Greeting Grandma’s five items can be entered through this form.
- i18n EN/FR.
- Lint/build pass.
