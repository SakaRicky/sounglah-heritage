# S023.14 - Lesson Item Renderer: CULTURAL_NOTE

## Epic

Epic 5 - Lessons and Learner Experience (parent: [S023](./00-overview.md))

## Status

Planned

## Goal

Warm culture card explaining why the lesson matters — no concept required.

## Dependencies

- S023.10 (player shell).

## UI Contract

- Card layout with earth-tone / cream Sounglah styling.
- Title from item (`Greeting Elders`).
- Body from `contentJson.noteTextEn` or `contentJson.noteTextFr` based on app UI locale (`en` / `fr`).

Example copy for Greeting Grandma item 5:

**EN:** In many families, greeting grandma or elders shows love, respect, and connection.

**FR:** Dans beaucoup de familles, saluer grand-mère ou les aînés montre l'amour, le respect et le lien.

- Optional image from `contentJson.imageUrl` + alt text.
- Primary action: **Got it** (i18n) — same as Continue in shell, or explicit button that calls advance.

Tone: respectful, family-centered, not lecture-heavy.

## Out Of Scope

- Long articles, external links, video.

## Acceptance Criteria

- CULTURAL_NOTE renders without concept link.
- Admin form requires both **Note text (English)** and **Note text (French)** fields.
- Got it / Continue advances player.
- Button chrome uses i18n; note body uses `noteTextEn` / `noteTextFr`.
- Lint/build pass.
